# ASSETS_NEEDED.md — Materiales a pedir al cliente

Lista para enviar al cliente. Marca lo recibido con [x].

## Marca
- [ ] **Logo** en buena calidad (preferible PNG con fondo transparente y/o SVG).
- [ ] Colores de marca (si los tiene).
- [ ] Tipografías de marca (si las tiene).

## Fotografía
- [ ] **Fotos horizontales** (para hero y secciones anchas).
- [ ] **Fotos verticales** (para móvil, tarjetas, galería).
- [ ] **Fotos del local** (fachada, interior, ambiente).
- [ ] **Fotos de productos / servicios** (platos, trabajos, resultados…).
- [ ] Foto del equipo / dueño (genera confianza), si quiere.

> Pedir la **mejor resolución posible**. Si las fotos son malas, sugerir sesión de
> fotos o usar placeholders y avisar de que la calidad final dependerá de las fotos.
> Confirmar **derechos de uso** de cualquier foto que no sea suya.

## Textos
- [ ] Descripción del negocio (1-2 párrafos).
- [ ] Lista de servicios / productos con breve descripción.
- [ ] Precios (si se van a mostrar).
- [ ] Frase/eslogan, si tiene.

## Confianza
- [ ] **Opiniones / testimonios** (o permiso para usar reseñas de Google).
- [ ] Años de experiencia, certificaciones, premios.

## Datos de contacto
- [ ] Teléfono.
- [ ] WhatsApp.
- [ ] Email.
- [ ] Dirección exacta.
- [ ] Horarios.

## Enlaces
- [ ] **Google Maps** (enlace o dirección exacta para el mapa).
- [ ] **Redes sociales** (Instagram, Facebook, TikTok…).
- [ ] Enlaces de reserva/pedido si usa plataformas (TheFork, Glovo, etc.).

## Legal (para publicar)
- [ ] Razón social / NIF para aviso legal.
- [ ] ¿Recoge datos? → política de privacidad y cookies.
